# CustomPlatControl
Platformer control remapping made simple.

## Features
- Remap Left, Jump, and Right for both players.
- Works only in Platformer mode.
- Classic mode controls are not changed.
- Your keybinds are saved in Geode settings.

## Default controls
- Player 1: `A` / `W` / `D`
- Player 2: `ArrowLeft` / `ArrowUp` / `ArrowRight`
